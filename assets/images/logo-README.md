# Dream Car Logo Files

Professional SVG logo designed with 2026 best practices.

## Files

- **logo.svg** — Full logo with tagline (280×80px viewBox)
- **logo-compact.svg** — Compact version without tagline (200×60px viewBox)
- **logo-icon.svg** — Icon mark for favicon/app icon (64×64px viewBox)

## Usage Guidelines

### Minimum Sizes
- Full logo: 180px width minimum
- Compact logo: 120px width minimum
- Icon mark: 32px × 32px minimum

### Clear Space
Maintain padding equal to 0.25× the logo height on all sides.

### Color Variants
Available via CSS classes:
- Default: Navy (#1A2B4A) + Orange (#FF6B35)
- Dark mode: `.logo--dark`
- Monochrome: `.logo--mono`
- Inverse: `.logo--inverse`

### Don'ts
- ❌ Don't stretch or distort aspect ratio
- ❌ Don't add effects (shadows, glows, outlines)
- ❌ Don't change colors without maintaining contrast
- ❌ Don't rotate (except icon for loading animations)
- ❌ Don't place on busy backgrounds without contrast

## Design Concept

Geometric "DC" monogram with parametric curves and negative space mastery. The interlocking letterforms create a circular motion path (wheel/rotation), while speed lines convey precision and momentum. Balances industrial strength with sophisticated geometry, reflecting Dream Car's 4.9-star reputation.
