## Highlights

- Vanilla HTML, CSS, and JavaScript (no framework overhead)
- Strong local SEO baseline (Open Graph + JSON-LD)
- Responsive and accessible UX (WCAG-minded semantic structure)
- Animated custom SVG brand mark
- Fast-loading, static-host friendly architecture

## Stack

- CSS3 with design tokens
- JavaScript (ES6+)

## Run Locally

Use any static server from this directory:

```bash
python3 -m http.server 4173
```

Then open http://localhost:4173.
